Code Change Report - 150226
==========================

Summary: Fixed "Missing Keys from Total War Attila" section in summary_log.txt showing CK3 MAA keys instead of Attila unit keys.

Bug Description:
The summary() function in cw_map_checker.py was using the wrong variable (missing_keys)
when printing the "Missing keys from Total War Attila" section. The variable missing_keys
contains CK3 Man-at-Arms keys missing from the mapper, not Attila unit keys. This caused
the Attila section to display identical output to the CK3 MAA section above it.

Changes:
--------

150226-00001  cw_map_checker.py:524 - Changed len(missing_keys) to len(missing_attila_keys) in the count display
150226-00002  cw_map_checker.py:525 - Changed range(0, len(missing_keys), ...) to range(0, len(missing_attila_keys), ...) in the iteration
150226-00003  cw_map_checker.py:526 - Changed missing_keys[i:i + output_columns] to missing_attila_keys[i:i + output_columns] in the row slicing
